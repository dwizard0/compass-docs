## Promotion Notes

- **O-1 through O-6** — Standard ship/regional workflow, RC approves in COMPASS
- **O-7 through O-9** — Require SFI Fleet Command approval; RC recommends to Fleet in COMPASS
- **O-10 and above** — Require Executive Committee vote; RC recommends to Fleet in COMPASS

!!! note
    For the authoritative rank structure, refer to the [SFI Member Handbook](https://sfi.org).
